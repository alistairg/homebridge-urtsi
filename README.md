# homebridge-urtsi

Somfy URTSI plugin for [HomeBridge](https://github.com/nfarina/homebridge)

This plugin makes use of the [node-urtsi](https://github.com/yungsters/node-urtsi) package.

# Configuration

```json
{
  "platform": "Somfy URTSI",
  "channels": [
    "Channel 1",
    "Channel 2",
    "Channel 3",
    "Channel 4",
    "..."
  ],
  "serialPath": "/dev/ttyUSB0"
}
```
